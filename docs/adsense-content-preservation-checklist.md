# AdSense 콘텐츠 보존 체크리스트

## 보존한 콘텐츠

- 서비스 목적 설명: 코트있음은 공개 예약 정보를 모아 보여주는 보조 서비스임을 하단 안내에 유지.
- 이용 방법: 지역, 날짜, 시간 선택 후 공식 예약처에서 최종 예약한다는 설명 유지.
- 지역별 코트 안내 링크: `/courts/` 유지.
- 데이터 출처와 한계 링크: `/data-source/` 유지.
- 이용 가이드 링크: `/guide/tennis-reservation/` 유지.
- 개인정보처리방침, 이용약관, 문의, robots.txt, sitemap.xml은 삭제하지 않음.

## 숨기지 않은 콘텐츠

- SEO/AdSense용 텍스트를 `display:none`, 화면 밖 이동, 투명 텍스트로 숨기지 않음.
- 사용자가 접근 가능한 하단 요약 블록과 기존 상세 페이지 링크로 제공.

## 광고 배치 원칙

- 필터, 날짜 버튼, 시간 칩, 공식 예약 버튼 내부/바로 옆에는 광고를 배치하지 않음.
- 광고 영역은 콘텐츠와 명확히 구분하고 `aria-label="광고"`를 사용.
- 광고 로드 후 레이아웃 이동을 막기 위해 사전 높이를 확보해야 함.
