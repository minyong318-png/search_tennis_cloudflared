⏳ 예약 정보를 불러오는 중입니다…

🔄 업데이트 방식 안내

이 페이지의 예약 정보는 일부 시설·날짜는 최신 정보가 아닐 수 있습니다.

✔ 데이터 수집 방식
• 용인시 공공예약 시스템을 주기적으로 확인하여 데이터를 갱신합니다.
• 용인시 테니스장은 당일 잔여 좌석을 00:00 기준 가장 가까운 갱신 시간 데이터를 남겨둠으로 정확한 정보를 보장하지 않습니다.
• 고양시 테니스장(고양테니스협회/성저)은 별도 수집 로직으로 갱신합니다.
✔ 업데이트 주기
• 최소 5분에 한번씩 업데이트를 진행하며 강제 업데이트 필요 시 🔄을 클릭하세요.
• 버튼 클릭 시 데이터 수집 시간인 약 1분 후 데이터가 업데이트 됩니다.
✔ 표시 시간 안내
• 상단의 “업데이트” 시간은 가장 최근에 데이터가 갱신된 시점입니다.
• 특정 시설·날짜의 최신 갱신 시점은 보장되지 않습니다.
✔ 알람 기능
• 사용자가 등록한 조건에 해당하는 새로운 예약 가능 시간이 발견되면 알림을 전송합니다.

🔔 알람 등록 유의사항

• “브라우저 → 홈 화면에 추가” 후 사용하셔야 알람을 받을 수 있습니다(인앱정책).
