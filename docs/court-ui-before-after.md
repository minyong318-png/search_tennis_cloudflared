# 코트있음 UI 변경 전후

## 변경 전

- 첫 화면에서 알림 등록 카드와 상세 설명이 예약 결과보다 먼저 크게 보였음.
- Supabase 공개 RPC 호출 후 도시별 갱신 시각 조회 4회가 추가로 발생했음.
- 용인시 결과에서 예약 가능한 시간이 없는 카드도 표시되어 목록이 길어졌음.

## 변경 후

- 첫 화면은 검색 조건과 예약 가능 결과에 집중.
- 결과 요약에서 현재 선택 조건의 가능 시간 수, 시설 수, 가장 빠른 시간을 바로 확인.
- 알림 등록은 접힘 영역으로 이동.
- 오늘/내일/모레 빠른 날짜 선택을 추가.
- 예약 가능 시간이 없는 시설 카드는 기본 결과에서 제외.
- 하단에 이용 방법, 지역별 안내, 데이터 출처 요약 블록을 유지.

## 클릭/스크롤 감소

- 날짜 선택: 달력 열기 전 오늘/내일/모레 1클릭 선택 가능.
- 예약 결과 탐색: 빈 시설 카드가 사라져 결과 스크롤 감소.
- 알림 기능: 기본 화면에서는 접혀 있고 필요할 때만 1클릭으로 열림.
